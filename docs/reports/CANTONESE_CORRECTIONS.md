## 如何新增校正規則

編輯 `src/pipeline/subtitle_pipeline_v2.py` 中嘅 `CANTONESE_CORRECTIONS` 字典：

```python
CANTONESE_CORRECTIONS = {
    "錯誤字詞": "正確字詞",
    # ...
}
```
