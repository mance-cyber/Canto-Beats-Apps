### 方案 A：等待（不推薦）

### 方案 B：取消並重新觸發（推薦）
1. 取消當前運行
2. 優化編譯配置（移除 LTO）
3. 重新推送 tag

### 方案 C：簡化編譯配置（最佳）
修改編譯腳本，減少編譯時間：
- 移除 `--lto=yes`（連結時優化）
- 改用 `--standalone`（不打包成單檔）
- 或只編譯 Windows（本地測試優先）
